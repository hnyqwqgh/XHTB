# 星河工具盒

星河工具盒是一款多功能实用工具应用，旨在为用户提供日常生活中所需的多样化工具服务。它包含了扫描文档、二维码生成、网络工具、文本转语音等多种功能，适用于 HarmonyOS 5 及以上设备。

## 功能特点

- **文档扫描**：通过设备摄像头快速扫描文档并进行 OCR 识别。
- **二维码生成与识别**：支持生成二维码和扫描识别二维码。
- **网络工具**：提供网络连接测试、IP 查询等网络相关功能。
- **文本转语音**：将文本内容转换为语音播放。
- **卡片管理**：提供便捷的卡片式操作界面，快速启动常用工具。

## 使用说明

1. **安装**
   - 在 HarmonyOS 设备的应用市场中搜索“星河工具盒”或通过以下链接下载安装：
     [应用市场链接](https://appgallery.huawei.com/app/detail?id=yylx.hny.qwq)

2. **启动应用**
   - 安装完成后，点击桌面图标启动应用。

3. **功能操作**
   - 主界面提供卡片式导航，点击相应功能进入对应页面。
   - 根据页面提示操作即可完成所需任务。

## 项目结构

- `entry/`：主模块目录，包含应用的主要代码和资源文件。
  - `src/main/ets/`：主要源码目录，包含多个功能模块的实现。
  - `resources/`：资源文件目录，包含图片、字符串、配置文件等。
- `AppScope/`：应用全局配置文件和资源目录。
- `hvigor/`：构建配置目录。

## 开发与构建

- 项目基于 HarmonyOS SDK 开发，使用 ETS (Enhanced TypeScript) 编写。
- 使用 HVIGOR 构建工具进行项目构建。
